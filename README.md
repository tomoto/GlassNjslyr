GlassNjslyr
===========

Sorry Japanese only.

習作のGoogle Glass用ハンズフリーニンジャスレイヤーリーダーです。

* "OK Glass, ゴウランガ!" でワザマエなスプラッシュとともに起動します。
* 頭を水平に回転してストーリーを選択し、オジギを決めて読み始めます。
  * タップは効きません。
* 謎のガイジンによる読み上げが行なわれます。
  * TTSが英語しか対応してないので本文をkakasiでローマ字に変換して強引に読ませています。
  * 熱くなります。
* ストーリーは選抜した5本を決め打ちしてあります。
